# SIAT Server

## Prerequisites
- [NestJS (Version 7.6)](https://docs.nestjs.com/#installation)

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

The application should be running locally at port **8000**.

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
